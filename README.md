# Neumorphism Loading Spinner - Soft UI Loader

Um componente de indicador de carregamento (Loader/Spinner) moderno e minimalista que explora a estética do Neumorfismo (Neumorphism / Soft UI). Desenvolvido exclusivamente com HTML5 e CSS3, o projeto destaca a aplicação de múltiplas sombras em direções opostas para criar um trilho circular tátil que parece esculpido diretamente sobre o plano de fundo, acompanhado por um ponto de progresso animado.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica simples do contêiner do carregador e do elemento de progresso.
* CSS3: Estilização completa, uso de formas circulares, paleta monocromática suave e gerenciamento de sombras projetadas complexas.
* Animações em CSS: Utilização de keyframes (`@keyframes`) para controlar a rotação contínua (`transform: rotate`) de forma otimizada para o navegador.

## Funcionalidades e Técnicas Aplicadas

* Estética Neumórfica Realista: O anel de fundo utiliza uma combinação precisa de sombras claras (brancas) e escuras (cinzas sutis) internas e externas (`box-shadow`), simulando um relevo tátil integrado perfeitamente à superfície.
* Ponto de Progresso de Alto Contraste: Inclusão de uma pequena esfera em azul vibrante que percorre o trilho circular, servindo como o elemento dinâmico de indicação de estado.
* Movimento Rotacional Infinito: Aplicação de uma animação contínua (`infinite`) com interpolação linear (`linear`) para garantir que o ponto azul orbite o anel de forma fluida e sem engasgos visuais.
* Centralização Absoluta com Flexbox: Alinhamento e posicionamento do componente exatamente no centro x/y da viewport sobre um fundo cinza claro fosco.
